How It Works

01
Raw Data
17 weather features collected daily across 36,854 USA locations from 2014–2025
CSV → 6.7M rows
02
75-Day Windows
Sliding 75-day windows per location. Fire doesn't happen in one day — dry conditions build up over weeks
HDF5 · 103GB
03
ConvLSTM2D
CNN detects feature patterns. LSTM remembers 75 days of history. Combined = ConvLSTM2D
50 Epochs · T4 GPU
04
Fire Probability
Model outputs 0–1 probability per location per day. Threshold 0.9 = fire alert
AUC 0.908
05
Trajectory Map
Wind physics simulation shows how fire spreads hour-by-hour over 24 hours
